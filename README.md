# 回忆博物馆 Memory Museum

一个 SillyTavern（酒馆）第三方扩展。把和角色的对话、句子、回忆，收进一座带音乐的小型「博物馆」。设计灵感来自旧情书拼贴美学——暖米白纸张、散落的标题、淡化的黑白照片。

---

## ✦ 功能一览

- **CD 悬浮球**：常驻屏幕的入口，可拖动。音乐播放时旋转（机臂固定不动），单击打开 / 关闭面板。
- **首页**：收藏集标题随机散落，装饰照片错乱重叠、随机淡化、可溢出面板边缘，带开场入场动画。
- **三大板块**（每个收藏集内）：文段品鉴 / 回忆薄 / 故事片段。三者都支持：
  - 添加文字 + **图片**
  - 每条都有**编辑 / 删除**图标
  - **批量删除**（点工具条的「批量管理」勾选多条）
  - **刷新键**（批量操作或卡顿后刷新状态）
- **顶栏 Album 播放器**：简约音乐栏，播放 / 暂停 / 下一首。
- **Album 音乐库**：导入音乐、设默认音乐、查看某首歌**跨收藏集关联**的所有内容并跳转。
- **音乐三层优先级**：条目绑定音乐 > 收藏集主题曲 > 默认音乐。
- **一键收藏**：聊天里 AI 的每条回复旁有 ♡，点一下收藏到「故事片段」。**收过一次就显示实心 ♥，再点即取消收藏**（一对一）。
- **返回键**：收藏集内部、Album 都有返回首页按钮；底部胶片条也能直接回首页 / 进 Album / 新建收藏集。

## ✦ 安装

1. 把整个 `Memory-Museum` 文件夹放进酒馆的：
   ```
   SillyTavern/data/<你的用户名>/extensions/third-party/Memory-Museum/
   ```
   （或用酒馆「扩展 → 安装扩展」填本仓库的 Git 地址）
2. 刷新酒馆页面。
3. 屏幕上会出现 CD 悬浮球；魔杖菜单里也有「回忆博物馆」入口。

> 目录层级要和示例一致（`extensions/third-party/Memory-Museum/`），否则 `index.js` 里对酒馆核心的相对导入路径会失效。

## ✦ 自定义（直接在 GitHub 改）

所有能改的地方都在代码里标了 `可修改` 注释，搜这两个字就能定位。常用的：

### 换素材 / 改尺寸 → `index.js` 顶部 `CONFIG`
```js
const CONFIG = {
    cdSkin:  '...png',      // CD 悬浮球图
    tonearm: '...png',      // 唱片机机臂
    background: '...jpg',   // 背景纹理
    decorImages: [ ... ],   // 首页装饰照片（任意增删）
    tabIcons: { quote:'...', memory:'...', story:'...' }, // 三个 tab 图标
    cdSize: 64,             // CD 悬浮球直径
    tabIconSize: 30,        // tab 图标尺寸（三个统一）
    decorOnScreen: 8,       // 开场散落照片数量
    heartEmpty: '♡', heartFull: '♥', // 收藏爱心符号
};
```

### 改配色 / 字体 → `style.css` 顶部 `:root`
```css
--mm-bg: #F5F0E8;   /* 背景 */
--mm-ink: #3D3229;  /* 文字 */
--mm-accent: #8B4D3B; /* 强调色 */
--mm-serif: 'Noto Serif SC', serif; /* 标题字体 */
```

### 其它常见调整
- 面板大小：`style.css` 搜 `.mm-dialog`（`width / height / max-width`）。
- 背景纹理可见度：`style.css` 搜 `.mm-bg-overlay` 的 `opacity`。
- 旋转速度、入场时长、呼吸幅度：`style.css` 搜 `mm-spin` / `mm-entrance` / `mm-breathe`。

## ⚠️ 关于「文件上传」（最可能需要按你的酒馆版本微调的点）

图片、音乐走酒馆的文件上传接口（`/api/files/upload`），只保存返回的路径，不把大文件塞进设置里。

不同酒馆版本接口可能略有差异。如果**图片 / 音乐上传不成功**，只需要改 `index.js` 里的一个函数：`uploadMedia()`（搜 `媒体上传`）。
- **图片**：上传失败会自动兜底为内嵌 base64，所以图片基本一定能用。
- **音乐**：文件较大，不做 base64 兜底，上传失败会提示，需要按你的版本调一下端点。

---

## ✦ 数据存储

所有数据存在酒馆的 `extension_settings['memory-museum']`，**全局保存**（不绑定某个角色或聊天），跟随酒馆设置体系。删除收藏集会级联删除其下条目；删除音乐会解除所有绑定。
